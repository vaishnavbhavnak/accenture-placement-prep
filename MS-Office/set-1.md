# Accenture MS Office MCQs – Set 1

Practice-oriented MCQs created for placement preparation.  
Questions are based on commonly asked patterns in Accenture assessments.

---

## Microsoft Word

1. **Starting with Microsoft Office 2003, Photo Editor was renamed to:**  
A. Photo Manager  
B. Picture Manager  
C. Picture Editor  
D. Paint Editor  

**Answer:** B

2. **Which feature is used to make selected text all capital or all small letters in Word?**  
A. Change Letter  
B. Change Sentence  
C. Change Case  
D. Change Word  

**Answer:** C

3. **In Word, what is the shortcut for centering selected text?**  
A. Ctrl + E  
B. Ctrl + R  
C. Ctrl + L  
D. Ctrl + J  

**Answer:** A

4. **What does the red wavy underline in Word indicate?**  
A. Formatting issue  
B. Spelling error  
C. Grammar suggestion  
D. Missing picture  

**Answer:** B

5. **Mail Merge in Word is used to:**  
A. Combine two documents  
B. Create personalized mass mails  
C. Encrypt document  
D. Insert a chart  

**Answer:** B

## Microsoft Excel

6. **What happens when you select a row and press Ctrl + Shift + Plus (+) in Excel?**  
A. Task Manager opens  
B. Numeric values are summed  
C. A new row is inserted  
D. New worksheet is added  

**Answer:** C

7. **What is the result of =AVERAGE(5,10,15)?**  
A. 5  
B. 10  
C. 15  
D. 20  

**Answer:** B

8. **Which Excel function returns TRUE only if all conditions are true?**  
A. =AVERAGEIFS  
B. =AND  
C. =CONCAT  
D. =COUNTA  

**Answer:** B

9. **Which shortcut copies the cell above to the current cell?**  
A. Ctrl + Shift + C  
B. Ctrl + Shift + "  
C. Alt + Shift + "  
D. Ctrl + Alt + "  

**Answer:** B

10. **What type of chart is best for showing parts of a whole?**  
A. Column  
B. Line  
C. Pie  
D. Bar  

**Answer:** C

11. **Which formula concatenates “MS” and “Office”?**  
A. =MERGE("MS","Office")  
B. =JOIN("MS","Office")  
C. =CONCATENATE("MS","Office")  
D. =ADD("MS","Office")  

**Answer:** C

12. **AutoFill in Excel is used to:**  
A. Merge cells  
B. Copy a pattern of data  
C. Format cells  
D. Delete duplicates  

**Answer:** B

## Common MS Office Fundamentals

**Answer:** B

A. Web Page  
B. Explorer

**Answer:** C

B. Outlook

A. Database suite  
B. Programming tool  
C. Productivity suite

A. Excel
