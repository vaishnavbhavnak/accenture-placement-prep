# Accenture MS Office MCQs – Set 2

Continuation of practice-oriented MCQs for Accenture placement preparation.

## Microsoft Word

1. **Which option is used to insert page numbers in Word?**  
A. Insert → Header  
B. Insert → Page Number  
C. Layout → Page Setup  
D. Review → Header  

**Answer:** B

2. **What is the default file extension of MS Word 2016?**  
A. .doc  
B. .txt  
C. .docx  
D. .rtf  

**Answer:** C

3. **Which shortcut is used to justify text in Word?**  
A. Ctrl + L  
B. Ctrl + E  
C. Ctrl + J  
D. Ctrl + R  

**Answer:** C

4. **Which view displays a document as it will appear when printed?**  
A. Draft View  
B. Web Layout  
C. Print Layout  
D. Outline View  

**Answer:** C

5. **Header and Footer appear in which area of a Word document?**  
A. Text area  
B. Margin area  
C. Body section  
D. Editing pane  

**Answer:** B

6. **Which feature allows checking synonyms of a word?**  
A. Spell Check  
B. Word Count  
C. Thesaurus  
D. Translate  

**Answer:** C

7. **Which tab contains the Find and Replace option?**  
A. Insert  
B. Home  
C. Review  
D. View  

**Answer:** B

9. **Which symbol is used before formulas in Excel?**

D. $

10. **Which function counts only numeric values?**

B. COUNT

11. **Which Excel feature removes duplicate records?**

B. Conditional Formatting

14. **Which shortcut edits the active cell?**
