# Accenture Computer Networking MCQs – Set 2

---

1. **Which OSI layer is responsible for logical addressing?**  
A. Data Link  
B. Network  
C. Transport  
D. Session  

**Answer:** B

---

2. **Which protocol resolves IP address to MAC address?**  
A. RARP  
B. DNS  
C. ARP  
D. ICMP  

**Answer:** C

---

3. **Which protocol resolves MAC address to IP address?**  
A. ARP  
B. RARP

**Answer:** B
