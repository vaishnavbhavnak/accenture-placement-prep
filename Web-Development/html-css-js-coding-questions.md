
# HTML, CSS & JavaScript Coding Questions (Accenture Practice)

This document contains **frequently asked HTML, CSS, JavaScript, and DOM coding questions**
for **Accenture and other service-based company assessments**.

Difficulty Level: Beginner to Intermediate (4–4.5 LPA roles)

---

## HTML + CSS CODING QUESTIONS

### 1. Create a Simple Registration Form

**Requirements**
- Fields: Name, Email, Password  
- Gender (radio buttons)  
- Skills (checkboxes)  
- Submit button  
- Use proper `<label>` tags  
- Password field should mask input  

**Skills Tested**
- `<form>`, `<input>`, `<label>`
- Radio & checkbox handling
- Basic layout

---

### 2. Design a Responsive Navbar Using CSS

**Requirements**
- Horizontal menu on desktop  
- Vertical menu on small screens  
- Use media queries  
- No JavaScript allowed  

**Skills Tested**
- Flexbox
- Media queries
- Responsive design basics

---

### 3. Create a Card Layout Using CSS

**Requirements**
- Display 3 cards in a row  
- Each card should have:
  - Image
  - Title
  - Description  
- On hover, card should slightly scale  

**Skills Tested**
- Flexbox / Grid
- `transform`, `transition`
- Box model

---

### 4. Center a Div Both Horizontally and Vertically

**Restriction**
- Use only CSS  

**Expected**
- Flexbox or Grid solution  

---

### 5. Create a Table With Alternate Row Colors

**Requirements**
- Header row styled differently  
- Even rows: light gray  
- Odd rows: white
