# HTML, CSS & JavaScript MCQs (Accenture Practice)

These MCQs are based on **actual Accenture MCQ patterns**.  
Difficulty: **Beginner / Fundamental**

## HTML MCQs

1. **Which tag is used to create a hyperlink in HTML?**  
A. `<link>`  
B. `<a>`  
C. `<href>`  
D. `<url>`  

**Answer:** B

2. **Which attribute is mandatory for the `<img>` tag?**  
A. src  
B. alt  
C. title  
D. width  

**Answer:** A

3. **Which input type hides the entered text?**  
A. text  
B. hidden  
C. password  
D. secure  

**Answer:** C

4. **Which tag is used to create an unordered list?**  
A. `<ol>`  
B. `<ul>`  
C. `<li>`  
D. `<list>`  

**Answer:** B

5. **Which HTML tag is used to create a form?**  
A. `<input>`  
B. `<form>`  
C. `<fieldset>`  
D. `<label>`  

**Answer:** B

6. **Which attribute specifies a unique identifier for an HTML element?**  
A. class  
B. name  
C. id  
D. type  

**Answer:** C

7. **Which tag is used for the largest heading?**  
A. `<h6>`  
B. `<head>`  
C. `<h1>`  
D. `<title>`  

**Answer:** C

8. **Which HTML tag is used to insert a line break?**  
A. `<br>`  
B. `<lb>`  
C. `<break>`  
D. `<hr>`  

**Answer:** A

9. **Which element is used to group form-related elements?**  
A. `<group>`  
B. `<form>`  
C. `<fieldset>`  
D. `<section>`  

**Answer:** C

10. **Which attribute opens a link in a new tab?**  
A. new  
B. open  
C. target="_blank"  
D. href="_new"  

**Answer:** C

## CSS MCQs

11. **Which CSS property is used to change text color?**  
A. bgcolor  
B. color  
C. font-color  
D. text-color  

**Answer:** B

12. **Which property is used to change background color?**  
A. color  
B. bgcolor  
C. background-color  
D. background  

**Answer:** C

13. **Which CSS display value hides an element completely?**  
A. visibility: hidden  
B. display: block  
C. display: none  
D. opacity: 0  

**Answer:** C

14. **Which unit is relative in CSS?**  
A. px  
B. cm  
C. em  
D. mm  

**Answer:** C

15. **Which property controls space inside an element?**  
A. margin  
B. padding  
C. border  
D. spacing  

**Answer:** B

16. **Which property is used to make a layout flexible?**  
A. position  
B. float  
C. display: flex  
D. overflow  

**Answer:** C

17. **Which selector selects an element with id="box"?**  
A. .box  
B. #box  
C. box  
D. *box  

**Answer:** B

18. **Which property is used to align text horizontally?**  
A. vertical-align  
B. text-align  
C. align  
D. position  

**Answer:** B

19. **Which CSS property adds shadow to text?**  
A. box-shadow  
B. shadow  
C. text-shadow  
D. font-shadow  

**Answer:** C

20. **Which CSS property is used for animation effects?**  
A. transform  
B. transition  
C. animation  
D. All of the above  

**Answer:** D

## JavaScript MCQs

21. **Which keyword is used to declare a variable in JavaScript?**  
A. var  
B. let  
C. const  
D. All of the above  

**Answer:** D

22. **Which symbol is used for single-line comments?**  
A. `<!-- -->`  
B. `#`  
C. `//`  
D. `/* */`  

**Answer:** C

23. **Which function displays a popup message?**  
A. prompt()  
B. confirm()  
C. alert()  
D. message()  

**Answer:** C

24. **Which operator checks both value and type?**  
A. ==  
B. =  
C. ===  
D. !=  

**Answer:** C

25. **Which method is used to access an element by id?**  
A. getElement()  
B. getElementById()  
C. querySelectorAll()  
D. getId()  

**Answer:** B

26. **Which event occurs when a button is clicked?**  
A. onload  
B. onhover  
C. onclick  
D. onpress  

**Answer:** C

27. **What is the output of `typeof null`?**  
A. null  
B. object  
C. undefined  
D. number  

**Answer:** B

28. **Which data type stores true or false?**  
A. String  
B. Number  
C. Boolean  
D. Object  

**Answer:** C

29. **Which loop executes at least once?**  
A. for  
B. while  
C. do-while  
D. foreach  

**Answer:** C

30. **Which JavaScript method converts string to integer?**  
A. Number()  
B. parseInt()  
C. toInt()  
D. convert()  

**Answer:** B
