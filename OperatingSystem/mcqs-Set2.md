# Accenture Operating System MCQs – Set 2

---

1. **Which type of OS supports time sharing?**  
A. Batch OS  
B. Distributed OS  
C. Time-Sharing OS  
D. Embedded OS  

**Answer:** C

---

2. **Which scheduling algorithm is non-preemptive by default?**  
A. Round Robin  
B. FCFS  
C. Preemptive SJF  
D. Multilevel Queue  

**Answer:** B

---

3. **Which OS concept allows multiple processes within a single program?**  
A. Multiprocessing  
B. Multitasking  
C. Multithreading  
D. Batch processing  

**Answer:** C

---

4. **Which memory allocation suffers from external fragmentation?**  
A. Paging  
B. Segmentation  
C. Virtual memory  
D. Cache memory  

**Answer:** B

---

5. **Internal fragmentation occurs in:**  
A. Segmentation  
B. Paging  
C. Swapping  
D. Compaction  

**Answer:** B

---

6. **Which of the following is NOT a CPU scheduling criteria?**  
A. Turnaround time  
B. Waiting time  
C. Response time  
D. Throughput  

**Answer:** D

---

7. **Which OS uses a command-line interface?**  
A. Windows  
B. UNIX  
C. macOS  
D. Android  

**Answer:** B

---

8. **Which of the following is a non-preemptive scheduling algorithm?**  
A. Round Robin  
B. Preemptive Priority  
C. FCFS  
D. Shortest Remaining Time  

**Answer:** C

---

9. **The maximum number of processes that can be in the ready state at once is limited by:**  
A. CPU speed  
B. Main memory  
C. Number of I/O devices  
D. Hard disk size  

**Answer:** B

---

10. **Which system call terminates a process?**  
A. fork()  
B. exec()  
C. wait()  
D. exit()  

**Answer:** D

---

11. **Which OS technique temporarily removes a process from memory?**  
A. Paging  
B. Swapping  
C. Segmentation  
D. Fragmentation  

**Answer:** B

---

12. **Which file access method accesses data sequentially?**  
A. Direct access  
B. Random access  
C. Sequential access  
D. Indexed access  

**Answer:** C

---

13. **Which OS structure combines the advantages of layered and monolithic design?**  
A. Monolithic  
B. Layered  
C. Microkernel  
D. Hybrid kernel  

**Answer:** D

---

14. **Which is NOT a type of kernel?**  
A. Monolithic  
B. Microkernel  
C. Hybrid  
D. Distributed  

**Answer:** D

---

15. **Which OS uses inode structure?**  
A. Windows  
B. UNIX  
C. DOS  
D. macOS only  

**Answer:** B

---

16. **Which condition allows resource preemption to prevent deadlock?**  
A. Mutual exclusion  
B. Circular wait  
C. No preemption  
D. Hold and wait  

**Answer:** C

---

17. **Which technique resolves deadlock by ignoring it?**  
A. Deadlock prevention  
B. Deadlock avoidance  
C. Deadlock detection  
D. Ostrich algorithm  

**Answer:** D

---

18. **Which algorithm is used in Banker’s Algorithm?**  
A. FCFS  
B. Deadlock avoidance  
C. Deadlock detection  
D. Paging  

**Answer:** B

---

19. **Which of the following best defines throughput?**  
A. Time taken to execute a process  
B. Number of processes completed per unit time  
C. Waiting time of processes  
D. CPU idle time  

**Answer:** B

---

20. **Which OS component handles memory allocation?**  
A. File system  
B. Scheduler  
C. Memory manager  
D. Device driver  

**Answer:** C

---

21. **Which scheduling algorithm may cause convoy effect?**  
A. FCFS  
B. Round Robin  
C. SJF  
D. Priority  

**Answer:** A

---

22. **Which is NOT a valid process synchronization tool?**  
A. Semaphore  
B. Mutex  
C. Monitor  
D. Cache  

**Answer:** D

---

23. **Which Linux command displays running processes?**  
A. ls  
B. ps  
C. pwd  
D. mkdir  

**Answer:** B

---

24. **Which OS allows tasks to meet strict deadlines?**  
A. Distributed OS  
B. Batch OS  
C. Real-Time OS

**Answer:** C

---

25. **Which type of OS is Android based on?**  
A. UNIX  
B. Linux  
C. Windows  
D. DOS  

**Answer:** B

---

26. **Which disk scheduling algorithm moves the disk arm in one direction only?**  
A. FCFS  
B. SSTF  
C. SCAN  
D. LOOK  

**Answer:** C

---

27. **Which OS feature ensures data consistency in concurrent access?**  
A. Scheduling  
B. Synchronization  
C. Paging  
D. Spooling  

**Answer:** B

---

28. **Which OS term refers to a set of pages in main memory?**  
A. Page table  
B. Working set  
C. Segment  
D. Frame  

**Answer:** B

---

29. **Which protection mechanism controls user access to resources?**  
A. Authentication  
B. Paging  
C. Scheduling

30. **Which OS function keeps track of file locations on disk?**  
A. Scheduler
